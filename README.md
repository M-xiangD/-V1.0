# 电脑电池健康检测程序

一个详细的Windows电脑电池健康检测工具，可以获取电池的详细信息、计算健康度，并提供专业的使用建议。支持命令行和Web界面两种查看方式。

## 功能特点

- **全面的电池信息获取**
  - 设计容量和当前完全充电容量
  - 电池循环次数
  - 制造商和序列号
  - 电池化学成分
  - 实时电量和状态

- **精确的健康度计算**
  - 基于实际容量与设计容量的比例计算
  - 考虑循环次数对健康度的影响

- **专业的健康评估**
  - 五级健康度评估（优秀、良好、一般、较差、极差）
  - 基于循环次数的使用建议
  - 个性化的维护建议

- **美观的Web界面**
  - 响应式设计，支持移动端
  - 可视化图表展示（饼图、柱状图）
  - 实时数据加载
  - 渐变色设计，视觉效果出色

- **报告保存功能**
  - 自动生成JSON格式的检测报告
  - 包含完整的电池信息和评估结果

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 方式一：Web界面查看（推荐）

#### Windows用户
双击运行 `run_web.bat` 文件，或在命令行执行：

```bash
python start_web_server.py
```

程序会自动：
1. 检测电池信息
2. 生成报告
3. 启动本地Web服务器
4. 在浏览器中打开报告页面

#### 手动启动
```bash
# 1. 先运行电池检测
python battery_health_checker.py

# 2. 启动Web服务器
python start_web_server.py
```

然后在浏览器中访问：`http://localhost:8000/index.html`

### 方式二：命令行查看

```bash
python battery_health_checker.py
```

### 程序化使用

```python
from battery_health_checker import BatteryHealthChecker

# 创建检测器实例
checker = BatteryHealthChecker()

# 运行检测并保存报告
checker.run(save_report=True)

# 获取电池信息
print(checker.battery_info)

# 获取健康度
print(f"健康度: {checker.health_score}%")

# 获取评估结果
print(f"评估: {checker.assessment}")
```

## 健康度评估标准

| 健康度范围 | 评估等级 | 说明 |
|-----------|---------|------|
| 90%以上 | 优秀 | 电池状态非常好 |
| 80%-90% | 良好 | 电池状态良好 |
| 60%-80% | 一般 | 容量有所下降 |
| 40%-60% | 较差 | 容量明显下降 |
| 40%以下 | 极差 | 严重老化，建议更换 |

## 循环次数参考

- **0-300次**：正常使用范围
- **300-500次**：电池已使用一段时间
- **500-800次**：接近使用寿命
- **800次以上**：已超过典型使用寿命

## 系统要求

- 操作系统：Windows 7及以上
- Python版本：Python 3.6及以上
- 需要管理员权限（用于powercfg命令）

## 注意事项

1. 程序需要管理员权限才能获取完整的电池信息
2. 首次运行会生成battery-report.html文件
3. 检测结果仅供参考，实际电池状态可能因使用环境而异
4. 建议定期检测以跟踪电池健康度变化

## 电池保养建议

1. **避免高温**：不要在高温环境下使用或充电
2. **适度充电**：避免长时间保持在100%电量
3. **避免深度放电**：电量低于20%时及时充电
4. **定期使用**：长期不使用时，保持50%左右的电量
5. **使用原装充电器**：使用官方认证的充电设备

## 故障排除

### 问题：无法获取电池信息
- 解决：以管理员身份运行程序
- 检查是否安装了wmi库：`pip install wmi`

### 问题：健康度显示为0
- 解决：确保电池已正确安装并被系统识别
- 重启电脑后再次运行程序

## 技术实现

程序通过以下方式获取电池信息：

1. **powercfg命令**：生成详细的电池报告HTML文件
2. **WMI接口**：获取实时的电池状态信息
3. **HTML解析**：从电池报告中提取关键数据
